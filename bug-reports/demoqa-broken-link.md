# Bug Report: Broken Link вызывает ошибку 500 Internal Server Error

## Окружение
- Браузер: Google Chrome 146.0.7680.178 (64 бит)
- ОС: Windows 11 x64 25H2
- Сайт: demoqa.com/broken

## Шаги воспроизведения
1. Открыть demoqa.com
2. Перейти в Elements → Broken Links - Images
3. Нажать "Click Here for Broken Link"

## Ожидаемый результат
Переход по ссылке открывает рабочую страницу с кодом 200

## Фактический результат
Страница возвращает ошибку 500 Internal Server Error

## Severity
Major

## Priority
Medium
